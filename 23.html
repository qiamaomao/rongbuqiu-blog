憨憨笔记 · Win10系统C语言环境搭建

原创
沂佰孜猫 憨憨笔记
2021年02月02日

此文章以记录为目的，边学边写，在WIN10上搭建C语言环境，有错的地方希望可以帮我纠正 。

MinGw的下载与安装

首先要知道它是干啥？ MinGw 百度百科

下载链接找到x86_64-posix-seh点击下载,把文件解压到自己感觉合适的地方。

  • MinGw下载 点击下载

  • MinGw安装(配置系统环境变量)

打开我的电脑-属性-高级系统设置-高级-环境变量
在系统变量中找到Path变量，选择编辑(没有则新建)，将之前mingw解压后目录下bin文件夹目录添加至其中

然后在系统变量中找到include变量(没有则新建一个，不区分大小写)，编辑，将原mingw解压目录下include文件夹目录添加进去。

配置好要重启一下电脑

编辑器选用 Visual Studio Code(vscode)

国内网络原因建议去腾讯软件中心下载 Visual Studio Code 下载

安装就下一步，下一步就好了，安装好打开，按照图里安装中文拓展，安装好下面有提示重新打开软件。

然后安装 c/c++ 和 code runner 和上图第二步一样搜索选第一个安装

创建第一个 C 程序 Hello.c

指定一个目录，新建文件夹，新建一个文件，保存文件后缀为.c在文件中写入

#include <stdio.h>
int main()
{
    /* 我的第一个 C 程序 */
    printf("Hello, World! \n");
    return 0;
}

点击右上角的三角形来运行它，在下方查看输出结果
